#点集拓扑 

>[!definition] 嵌入
>设$X,Y$都是[[拓扑空间]], $f:X\to Y$是一个[[连续映射|连续]]的[[映射|单射]]. 将$X$在$f$下的像$f(X)$看作$Y$的一个子空间, 并且限制$f$的值域到$f(X)$得到的双射$f':X\to f(X)$是一个[[同胚]], 则称$f:X\to Y$是一个**拓扑嵌入**(topological imbedding), 或称为从$X$到$Y$中的一个**嵌入**(imbedding).

>[!hint] 提示
>概括地说, 满足以下三个条件的连续映射就是一个嵌入:
>1. 单射;
>2. 连续;
>3. 限制到双射后是同胚.
>
>>[!question] 为什么要限制$f'$是同胚?
>>我们来看一个经典反例: $f:[0,1)\to\R^2$.
>>我们规定$f(t)=(\cos(2\pi t),\sin(2\pi t))$, 这是从$\R$上区间到$\R^2$上单位元$S^1$的映射.
>>```tikz
>>\begin{document}
>>\begin{tikzpicture}[domain=-5:5,scale=1]
>>\draw[very thin,color=gray] (-1.4,-1.4) grid (1.4,1.4);
>>\draw[->] (-1.6,0) -- (1.6,0) node[right] {};
>>\draw[->] (0,-1.6) -- (0,1.6) node[above] {};
>>\draw[color=red,thick] (0,0) circle (1);
>>\draw[color=blue,thick] (0,0) -- (1,0);
>>%\draw[color=orange] plot (\x,{0.05*exp(\x)}) node[right] {};
>>\end{tikzpicture}
>>\end{document}
>>```
>>
>> 显然, $f$符合单射和连续的条件, 然而他的限制$f':[0,1)\to S^1$却不是同胚, 例如对于$U=[0, \frac{1}{4})$, 它对应的圆弧如下图所示:
>> ```tikz
>>\begin{document}
>>\begin{tikzpicture}[domain=-5:5,scale=1]
>>\draw[very thin,color=gray] (-1.4,-1.4) grid (1.4,1.4);
>>\draw[->] (-1.6,0) -- (1.6,0) node[right] {};
>>\draw[->] (0,-1.6) -- (0,1.6) node[above] {};
>>\draw[color=red!50!black,thick] (0,0) circle (1);
>>\draw[color=red!75!white,ultra thick] (1,0) arc (0:90:1);
>>\draw[color=blue!50!black,thick] (0,0) -- (1,0);
>>\draw[color=blue!75!white,ultra thick] (0,0) -- (0.25,0);
>>%\draw[color=orange] plot (\x,{0.05*exp(\x)}) node[right] {};
>>\node[right, below] at (1,0) {$f(0)$};
>>\end{tikzpicture}
>>\end{document}
>>```
>>虽然该集合在$[0,1)$上是开集, 但$f(U)$却不是单位元上的的开集, 这是因为不存在$\R^2$中$f(0)$的邻域$V$, 使得$V\cap S^1\subset f(U)$.
>
>之后我们会讨论, 圆实际上是一维的流形, 但它无法嵌入$\R$.
