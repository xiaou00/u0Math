#范畴论 

下面列举了一些常见的[[范畴]]:

| 范畴对象        | 符号           | 态射     | 同构     |     |
| ----------- | ------------ | ------ | ------ | --- |
| 集合          | $\sf{Set}$   | 映射     | 双射     |     |
| 群           | $\sf{Grp}$   | 群同态    | 群同构    |     |
| 拓扑空间        | $\sf{Top}$   | 连续映射   | 同胚     |     |
| 环           | $\sf{Rng}$   | 环同态    | 环同构    |     |
| 域           | $\sf{Field}$ | 域同态    | 域同构    |     |
| Abel群       | $\sf{AbGrp}$ | 群同态    | 群同构    |     |
| $R$-模       | $\sf{Mod}_R$ | 模同态    | 模同构    |     |
| 线性空间        | $\sf{Vect}$  | 线性映射   | 线性同构   |     |
| 赋范空间        | $\sf{Norm}$  | 有界线性算子 |        |     |
| Banach空间    | $\sf{Ban}$   | 有界线性算子 |        |     |
| 一致空间        | $\sf{Unif}$  | 一致连续映射 | 一致连续同胚 |     |
| 度量空间        | $\sf{Met}$   | 等距嵌入   | 等距同构   |     |
| 拓扑群         | $\sf{TpGrp}$ | 连续群同态  | 连续群同构  |     |
| 关系          | $\sf{Rel}$   | 二元关系   |        |     |
| Hausdorff空间 | $\sf{Haus}$  | 连续映射   | 同胚     |     |
