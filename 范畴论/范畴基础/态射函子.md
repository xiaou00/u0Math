>[!definition] 态射函子
>设$\frak C$是一个[[局部小范畴]], 其**态射函子**(hom-functor)是指二元函子:
>$${\frak C}(-,-):\frak C^{\rm op}\times\frak C\to\sf Set$$
>他将对象$x,y\in\frak C$映射到对应的态射集${\frak C}(x,y)$. 对于固定的$x\in\frak C$, 能得到固定一个变元的态射函子:$$\begin{aligned}
{\frak C}(-,x):&{\frak C}^{\rm op}\to\sf Set\\
{\frak C}(x,-):&{\frak C}\to\sf Set
\end{aligned}$$
>若固定其中一边, 令$A\in\ob C$, ${\frak C}(A,-):\frak C\to\sf Set$定义如下:$$\begin{aligned}
B&\mapsto{\frak C}(A,B)\\
(f:B\to C)&\mapsto({\frak C}(A,f):g\mapsto fg)
\end{aligned}$$
>称之为**关于$A$的态射函子**. 对偶地可以定义${\frak C}(-,A):\frak C^{\rm op}\to\sf Set$, 称之为**关于$A$的反变态射函子**.

如上图所示, Hom函子的作用就是将局部小范畴对于的元素映射到其态射集, 将其态射映到对应的复合$g\mapsto f\circ g$. 这里的对象转换关系需要额外注意:

| 元素                  | 类型               |
| ------------------- | ---------------- |
| $f$                 | 态射               |
| $\frak C$           | 范畴               |
| $\mathfrak{C}(A,-)$ | 函子               |
| $\mathfrak{C}(A,X)$ | 集合, 元素为态射        |
| $\mathfrak{C}(A,f)$ | 映射($\sf Set$中态射) |
这样我们就自然地引出了$\mathfrak{C}(-,-)$. 它是$\frak C^{\rm op}$到函子范畴$[{\frak C},{\sf Set}]$的反变函子.
